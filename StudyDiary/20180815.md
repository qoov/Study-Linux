# Just start linux study.


## Study Set up
- Install Docker
- Install Linux Container - Ubuntu
~~~
docker run ubuntu // It's pull latest ubuntu
docker run --rm -it ubuntu /bin/bash
/# cat /etc/issue
~~~
- Install Linux Container - CentOS
~~~
docker run centos // It's pull latest centos
docker run --rm -it ubuntu /bin/bash
/# cat /etc/issue
~~~
